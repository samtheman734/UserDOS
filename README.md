# UserDOS

**UserDOS** is a Python-based MS-DOS style command prompt emulator. It simulates a simple DOS environment where you can navigate directories, manage files, and run basic commands.

---

## Features

- Command prompt `A:\>`
- Commands: `HELP`, `DIR`, `COPY`, `DEL`, `REN`, `TYPE`, `CLS`, `EXIT`, `EDIT`, `COPY CON`
- Virtual directories: `general`, `documents`, `home`
- Create, delete, copy, and view text files

---
